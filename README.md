Projeto de ETL desenvolvido no programa Santander 2025 – Ciência de Dados com Python (DIO).

O projeto consiste em um notebook em Python que realiza um fluxo de ETL com dados fictícios, organizando as informações em um JSON padronizado com o apoio do Gemini (Google) para formatação e estruturação do conteúdo.

O objetivo é demonstrar, de forma prática, as etapas do processo:

- Criação de uma entrada de dados inventada;

- Envio do conteúdo ao Gemini para padronização;

- Obtenção de uma resposta estruturada (JSON);

- Salvamento e reutilização do JSON gerado.
